Acerca de mi

Estas son algunas de las tecnologías que aprendí a manejar durante mis estudios como autodidacta mediante cursos web y leyendo documentación de las librerías.

HTML y CSS

Lo primero fue aprender HTML para conocer como es la estructura de la web. Seguido de esto la forma en que se le da una apariencia y buena presentación a esta, aprendiendo desde lo básico de CSS hasta el manejo de flexbox, grid y media queris para lograr páginas que se adapten a cualquier dispositivo

Sass

Lo siguiente fue aprender a manejar un procesador de CSS para realizar de manera más rápida y eficiente la creación de estilos, para esto elegí Sass

JavaScript

Para lograr interactuar con las aplicaciones web fue necesario aprender JavaScript, desde las funciones para interactuar con el DOM hasta la creación y forma de interactuar con los object y arrays

TypeScript

El familiarizarme con TypeScript fue fundamental para lograr un código más organizado y así evitar muchos errores de tipado. También fue parte importante para empezar a aprender un framework

Angular

Angular es el framework que decidí aprender al sentirme cómodo viendo su estructura. Pasando desde la creación de componentes y rutas hasta la creación de servicios, el manejo de observables(RxJS) y la creación de un store(NgRx) para un mejor manejo de la data
