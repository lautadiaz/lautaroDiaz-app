Acerca de mi

Estas son algunas de las tecnologias que aprendi a manejar durante mis estudios como autodidacta mediante cursos web y leyendo documentacion de las librerias.

HTML y CSS

Lo primero fue aprender HTML para conocer como es la estructura de la web. Seguido de esto la forma en que se le da una apariencia y buena presentacion a esta aprendiendo desde lo basico de CSS hasta el manejo de flexbox, grid y media queris para lograr paginas que se adapten a cualquier dispositivo

Sass

Lo siguiente fue aprender a manejar un procesador de CSS para realizar de manera mas rapida y eficiente la creacion de estilos, para esto elegi Sass

JavaScript

Para lograr interactuar con las aplicaiones web fue necesario aprender JavaScript, desde las funciones para interactuar con el DOM hasta la creacion y forma de interactuar con los object y arrays

TypeScript

El familiarizarme con TypeScript fue fundamental para lograr un codigo mas organizado y asi evitar muchos errores de tipado. Tambien fue parte importante para empezar a aprender un framework

Angular

Angular es el framework que decidi aprender al sentirme comodo viendo su estructura. Pasando desde la creacion de componentes y rutas hasta la creacion de servicios, el manejo de observables(RxJS) y la creacion de un store(NgRx) para un mejor manejo de la data
